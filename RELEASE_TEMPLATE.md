# Release Template

## Evo Chatterbox Portable for Windows

Gerador local de audio com interface web e inicializacao simples por `INICIAR.bat`.

### Inclui

- interface web pronta para uso
- deteccao automatica de GPU NVIDIA
- fallback para CPU
- suporte a multiplos idiomas
- presets de emocao
- documentacao da API FastAPI
- colecao Postman

### Como usar

1. Instale o Python 3.11.
2. Extraia o ZIP.
3. Execute `INICIAR.bat`.
4. Aguarde a abertura do navegador.

### Observacoes

- no primeiro uso, o sistema baixa dependencias e modelos
- os caches ficam dentro da pasta do projeto
- para encerrar, feche a janela `Evo Chatterbox Server`
